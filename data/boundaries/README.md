# Hong Kong Administrative Boundaries

`hksar_18_district_boundary.json` is the official 18-district administrative
boundary dataset published by the Home Affairs Department of the HKSAR
Government.

- Source: https://data.gov.hk/en-data/dataset/hk-had-json1-hong-kong-administrative-boundaries
- Download URL: https://www.had.gov.hk/psi/hong-kong-administrative-boundaries/hksar_18_district_boundary.json
- Downloaded: 2026-06-15

The dashboard uses the district polygons both for server-side point-in-polygon
filtering and for the boundary layer displayed on the map.
